# كيفية تشغيل السيرفر

## الخطوة 1: التأكد من MongoDB
- إذا كان MongoDB مثبت محلياً، تأكد أنه يعمل
- أو استخدم MongoDB Atlas (رابط الاتصال من ملف `.env`)

## الخطوة 2: تثبيت المكتبات (إذا لم تكن مثبتة)
```bash
cd server
npm install
```

## الخطوة 3: تشغيل السيرفر

### للتطوير (مع إعادة التشغيل التلقائي):
```bash
npm run dev
```

### للإنتاج:
```bash
npm start
```

السيرفر سيعمل على: `http://localhost:4000`

---

## لتشغيل الفرونت إند أيضاً:

افتح نافذة Terminal جديدة:
```bash
cd ..
npm install
npm start
```

الفرونت إند سيعمل على: `http://localhost:3000`
